Instructions

To edit code, type in your name and press 'Online code editor'. If you wrote web code (e.g. HTML, JavaScript, CSS), duplicate this tab and press 'View your files' and press on the file you created.

If you wrote a backend (e.g. Rust, NodeJS) type in the port you bound in your code under 'Backend Services' and press 'Go to your server'.
